Callbacks

Funções chamadas automaticamente pelo engine

Ciclo de Vida
onCreate()
Início da criação do estado. Use para criar sprites e variáveis.
onCreatePost()
Após tudo criado. Personagens e strums já existem aqui.
onUpdate(elapsed)
Todo frame. elapsed = segundos desde o último frame.
onUpdatePost(elapsed)
Após o update do PlayState.
onDestroy()
Quando o estado é destruído.
Ritmo & Música
onBeatHit()
Cada beat (usa curBeat). 4 beats por seção.
onStepHit()
Cada step (usa curStep). 16 por seção.
onSectionHit()
Cada seção.
onCountdownStarted()
Countdown iniciado. Strums têm posição aqui.
onCountdownTick(counter)
counter: 0=Three, 1=Two, 2=One, 3=Go!, 4=Start.
onSongStart()
Música começa a tocar.
onEndSong()
Música terminou. Retorne Function_Stop para cutscene.
Notas
goodNoteHit(id,dir,type,sustain)
Jogador acertou uma nota.
opponentNoteHit(id,dir,type,sustain)
Oponente acertou uma nota.
noteMiss(id,dir,type,sustain)
Jogador errou uma nota.
onSpawnNote(id,dir,type,sustain)
Uma nota aparece na tela.
Function Returns
ValorEfeito
Function_StopCancela o comportamento padrão
Function_ContinueContinua normalmente
Function_StopLuaImpede chamada nos próximos scripts Lua
Function_StopHScriptImpede chamada nos próximos HScripts
Function_StopAllImpede em todos os scripts

Tweens & Timers

Animações de propriedades e delays

doTween
doTween('bfMove', 'boyfriend', {x=600}, 1, 'linear')
doTween('zoom', 'camGame', {zoom=1.5}, 0.5, 'sineOut')
doTween('fade', 'mySprite', {alpha=0}, 0.8, 'cubeInOut')
cancelTween('bfMove')
function onTweenCompleted(tag)
  if tag == 'fade' then removeLuaSprite('mySprite') end
endlua
Easings: linearsineInsineOutsineInOutcubeIncubeOutbounceOutelasticOutbackOut
runTimer
runTimer('delay', 2)
runTimer('pulse', 0.5, 0)
function onTimerCompleted(tag, loops, loopsLeft)
  if tag == 'delay' then startCountdown() end
endlua

Beat sprite — HScript HX
import flixel.FlxSprite;
import flixel.tweens.FlxTween;
import states.PlayState;
var spr:FlxSprite;
function onCreate() {
  spr = new FlxSprite(100,100);
  spr.loadGraphic(Paths.image('mySprite'));
  PlayState.instance.add(spr);
}
function onBeatHit() {
  spr.scale.set(1.2,1.2);
  FlxTween.tween(spr.scale,{x:1,y:1},0.2);
}hscript
