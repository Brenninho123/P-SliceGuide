hscript
P-Slice vs Psych puro: P-Slice herda toda a API Lua/HScript do PE 1.0. Scripts feitos pra PE 1.0 funcionam no P-Slice — mas o P-Slice adiciona variáveis extras de V-Slice (ex: freeplay metadata).
Project.xml: Certifique que LUA_ALLOWED e HSCRIPT_ALLOWED estão definidos no Project.xml do P-Slice. Sem eles, os scripts são ignorados silenciosamente.

Callbacks

Funções chamadas automaticamente pelo engine em resposta a eventos do jogo

Ciclo de Vida
onCreate()
Início da criação do estado. Use para criar sprites e inicializar variáveis.
onCreatePost()
Após tudo criado. Personagens e strums já existem aqui.
onUpdate(elapsed)
Todo frame. elapsed = tempo desde o último frame em segundos.
onUpdatePost(elapsed)
Após o update do PlayState. Mais seguro para modificar posições.
onDestroy()
Quando o estado é destruído (fim de música ou saída).
Ritmo & Música
onBeatHit()
Cada beat (usa curBeat). 4 beats por seção.
onStepHit()
Cada step (usa curStep). 4 steps por beat = 16 por seção.
onSectionHit()
Cada seção (usa curSection).
onCountdownStarted()
Countdown iniciado. Strums já têm posição aqui.
onCountdownTick(counter)
counter: 0=Three, 1=Two, 2=One, 3=Go!, 4=Start.
onSongStart()
Música começa de fato a tocar.
onEndSong()
Música terminou. Retorne Function_Stop para fazer cutscene.
Notas
goodNoteHit(id, direction, noteType, isSustainNote)
Jogador acertou uma nota.
opponentNoteHit(id, direction, noteType, isSustainNote)
Oponente acertou uma nota.
noteMiss(id, direction, noteType, isSustainNote)
Jogador errou uma nota.
noteMissPress(direction)
Jogador pressionou tecla sem nota (ghost tap desativado).
onSpawnNote(id, direction, noteType, isSustainNote)
Uma nota aparece na tela.
Inputs do Jogador
onKeyPress(key)
key: 0=esq, 1=baixo, 2=cima, 3=dir.
onKeyRelease(key)
Tecla soltada.
Eventos & Estado
onEvent(name, val1, val2, strumTime)
Um evento do chart foi ativado.
onGameOver()
Jogador morreu. Retorne Function_Stop para cancelar.
onPause()
Jogo pausado.
onResume()
Jogo resumido.
onRecalculateRating()
Rating sendo recalculado. Retorne Function_Stop para customizar.
Function Returns — controle de fluxo
-- Lua
function onEndSong()

  return Function_Stop

end

// HScript

hscript
ValorEfeito
Function_StopCancela o comportamento padrão (ex: impede fim de música)
Function_ContinueContinua normalmente (padrão)
Function_StopLuaImpede que o callback rode nos próximos scripts Lua
Function_StopHScriptImpede que rode nos próximos HScripts
Function_StopAllImpede em todos os scripts seguintes

Tweens & Timers

Animações de propriedades e delays com easing

doTween
-- doTween(tag, objeto, propriedades, duração, easing)
doTween('bfMove', 'boyfriend', {x=600, y=300}, 1, 'linear')
doTween('zoomIn', 'camGame', {zoom=1.5}, 0.5, 'sineOut')
doTween('fadeOut', 'mySprite', {alpha=0}, 0.8, 'cubeInOut')

-- Cancelar tween
cancelTween('bfMove')

-- Callback ao terminar — onTweenCompleted(tag)
function onTweenCompleted(tag)
  if tag == 'fadeOut' then
    removeLuaSprite('mySprite')
  end
end

Timers
-- runTimer(tag, tempo, loops)
runTimer('delay', 2)           -- 2 segundos, 1 vez
runTimer('pulse', 0.5, 0)     -- loop infinito

-- Callback — onTimerCompleted(tag, loops, loopsLeft)

  if tag == 'delay' then

  end

HScript no P-Slice: Usa SScript (HScript-Iris). Você tem acesso a praticamente todas as classes do engine — FlxSprite, FlxTween, FlxTimer, Paths, ClientPrefs, PlayState, etc. Muito mais poderoso que Lua para mecânicas complexas.

import flixel.FlxSprite;
import flixel.tweens.FlxTween;
import states.PlayState;

var game = PlayState.instance;
var beatSprite:FlxSprite;

function onCreate() {
  beatSprite = new FlxSprite(100, 100);
  beatSprite.loadGraphic(Paths.image('mySprite'));
  beatSprite.cameras = [game.camHUD];
  game.add(beatSprite);
}

function onBeatHit() {
  beatSprite.scale.set(1.2, 1.2);
  FlxTween.tween(beatSprite.scale,
    {x: 1, y: 1},
    0.2,
    {ease: flixel.tweens.FlxEase.sineOut});
}
